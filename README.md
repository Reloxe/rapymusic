# rapymusic
Free Open Source Discord Music Bot
## Setting Up

1- Download Nodejs

2- Download Python (Add To Path Check!)

3- Download Visual Studio and install C++ Desktop Development

4- run npm i

5- Open the index.js file and change your discord bot token in line 105, line 65. And line 71 bot id changes your bot id.

6- run start.bat or start.sh


## Enjoy.
